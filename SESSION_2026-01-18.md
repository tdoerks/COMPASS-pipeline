## Technical Notes

### FastTree vs IQ-TREE
- IQ-TREE not available on Beocat
- FastTree is scientifically rigorous alternative
- GTR+Gamma model comparable to IQ-TREE
- 1000 bootstrap replicates for branch support
- Much faster (<1 min vs hours for tree building)

### Phylogeny Workflow
1. Extract prophages from VIBRANT output
2. Label with year/sample ID
3. Subsample to 500 if too many sequences
4. MAFFT alignment (longest step: ~12 hours for 500 seqs)
5. FastTree phylogeny (fast: <1 minute)
6. Output: Newick tree + metadata TSV

### AMR Detection Methods
- **Method 1**: Coordinate-based (has bug, finds 0 genes)
- **Method 2**: Annotation search (limited sensitivity)
- **Method 3**: Direct AMRFinder scan (GOLD STANDARD - definitive)
