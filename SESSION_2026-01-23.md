## New Feature: E. coli 2025 Monthly Sampling

Created `run_ecoli_2025_monthly_10.sh` - stratified monthly sampling script

### What it Does

1. **Downloads** all E. coli metadata for 2025 from NCBI SRA
2. **Filters** for:
   - Illumina platform only
   - GENOMIC library source (isolates, not metagenomics)
3. **Stratifies by month** and randomly selects 10 samples per month
4. **Total**: 120 samples (12 months × 10 samples)
5. **Runs** COMPASS pipeline on selected samples

### Features

- **Temporal coverage**: Even distribution across all 12 months of 2025
- **Random sampling**: Reproducible (seed=42)
- **Adaptive**: If any month has <10 samples, selects all available
- **Metadata preservation**: Saves which samples selected from each month

### Usage

```bash
cd /fastscratch/tylerdoe/COMPASS-pipeline
git pull origin v1.3-dev
sbatch run_ecoli_2025_monthly_10.sh
```

### Output

- Results: `/fastscratch/tylerdoe/ecoli_2025_monthly_YYYYMMDD/`
- Metadata: `metadata/ecoli_2025_metadata.csv` (all available)
- Filtered: `metadata/ecoli_2025_monthly_filtered.csv` (selected 120)
- SRR list: `metadata/srr_accessions.txt`
